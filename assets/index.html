Seletores CSS

Bem-Vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo:Busca elementos por uma tag HTML
  • Seletor por ID(#):Busca elementos atráves de atributo "id"
  • Seletor por Classe(.):Busca elementos atráves de atributo "class"
  • Seletor por Universal(*):Seleciona todos os elementos do HTML
  • Seletor por Atributo ([atrib]):Seleciona elementos que possuem um atributo especifíco em nosso documento HTML e conseguimos também buscar atributos com um valor especifico
